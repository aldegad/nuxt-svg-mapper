#svg-mapper 기능 목록

## 시작 방법

1. 프로젝트 생성
2. 프로젝트를 클릭하여 프로젝트로 들어감
3. 프로젝트 내에서 캔버스 생성
4. 캔버스 내에서 화면 상단의 이미지 업로드 클릭
5. 사용

## 추가 논의에서 나온 사항

- 펜툴 구현되어야함(ㄱ자 모양의 공간같은 것 있음)
- 그룹 구현 더 편하게 및 그룹 기능 고도화 필요
- 도면안에 도면이 있을 수 있음
- 뷰어에서는 도면안의 도면 클릭 시, 확대되는 기능
- 툴팁 및 모든 텍스트 한글로
